Select the useful flotting blocks for the local data of the current processor. parameters (idpnew, local_V_size_new, nnew) for the computation. ide the local range are set with a size nnew equal to zero. This compute the right parameters (idpnew, local_V_size_new, nnew) for the computation. All the block outside the local range are set with a size nnew equal to zero. local_V_size_new correspond to the size without the shift between the global rank index and the global index of the first flotting block. idnew is then set to the index of this first flotting block.
